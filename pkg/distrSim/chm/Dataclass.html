Generates an object of class "Dataclass".

Dataclass(Data, ...)

Arguments

Data the data to be filled in
... additional parameters.
filename Object of class "character": the filename the data shall be saved
name Object of class "character": a name for the Data
Obsdimnames optional names for the observation dimensions
Runnames optional names for the runs
