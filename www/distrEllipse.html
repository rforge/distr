What is "distrEllipse" meant for?

The aim of package "distrEllipse" is to provide support for elliptically symmetric distributions in the framework of package "distr" .


Manual

for versions of the illustrations prior to 1.8, a somewhat more detailed manual to this package in pdf-format is available here;
from version 1.8 on, we have converted this manual into a common vignette to packages distr, distrEx, distrEllipse, distrSim, distrTEst, distrMod, distrTeach,
which is available in the mere documentation package distrDoc. To use it you may type
        require("distrDoc"); V<-vignette("distr"); print(V); edit(V)

Download

Windows

  • use
    install.packages("distrEllipse", repos = "R-Forge.R-project.org")
  • to be removed by 
    remove.packages("distrEllipse")
  • to be used by 
    library("distrEllipse")
    or 
    require("distrEllipse")

Linux

  • use
    install.packages("distrEllipse", repos = "R-Forge.R-project.org")
  • to be removed by
    R CMD REMOVE distrEllipse
  • to be used as under Windows
