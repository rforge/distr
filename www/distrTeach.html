What is "distrTeach" meant for?

The aim of package "distrTeach" is to provide illustrations based on package "distr" to topics in Stochastics and Statistics taught in secondary school.
Up to now, we have moved the corresponding illustrations  for the CLT and the LLN from package "distrEx" to this package.


Manual

for versions of the illustrations prior to 1.8, a somewhat more detailed manual to this package in pdf-format is available here;
from version 1.8 on, we have converted this manual into a common vignette to packages distr, distrEx, distrSim, distrTEst, distrMod, distrTeach,
which is available in the mere documentation package distrDoc. To use it you may type
        require("distrDoc"); V<-vignette("distr"); print(V); edit(V)

Download

Windows

  • use
    install.packages("distrTeach", repos = "R-Forge.R-project.org")
  • to be removed by 
    remove.packages("distrTeach")
  • to be used by 
    library("distrTeach")
    or 
    require("distrTeach")

Linux

  • use
    install.packages("distrTeach", repos = "R-Forge.R-project.org")
  • to be removed by
    R CMD REMOVE distrTeach
  • to be used as under Windows
