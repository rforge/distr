What is "distrDoc" meant for?

The aim of package "distrDoc" is to provide a common documentation in form of a vignette "distr" to the distrXXX-package family.
Currently this family comprises packages distr, distrEx, distrSim, distrTEst, distrMod, distrTeach,
You may use it e.g. as:
        require("distrDoc"); V<-vignette("distr"); print(V); edit(V)

Download

Windows

  • for version >= 2.0 use
    install.packages("distr", repos = "R-Forge.R-project.org")
  • as Win-Zip-File for R >=2.2.0 (Version 1.9)
  • as Win-Zip-File for R >=2.2.0 (Version 1.8)
  • to be installed by
    • the R-gui File picker [Packages ->Install package(s) from local zip-files...] or 
    • install.packages("distrDoc", <path to the file distrDoc.zip in your File system>, NULL)
  • to be removed by 
    remove.packages("distrDoc")
  • to be used by 
    library("distrDoc")
    or 
    require("distrDoc")

Linux

  • for version >= 2.0 use
    install.packages("distr", repos = "R-Forge.R-project.org")
  • as Linux-tar-gz-File for R >=2.2.0 (Version 1.9)
  • as Linux-tar-gz-File for R >=2.2.0 (Version 1.8)
  • to be installed by
    R CMD INSTALL distrDoc_1.x.tar.gz

  • to be removed by
    R CMD REMOVE distrDoc
  • to be used as under Windows
