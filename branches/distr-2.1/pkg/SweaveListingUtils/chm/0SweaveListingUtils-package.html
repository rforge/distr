Package SweaveListingUtils

Description

SweaveListingUtils contains provides utilities for defining R / Rd as TeX-package-listings "language" and including R / Rd source file (sniplets) copied from R-forge in its most recent version thereby avoiding inconsistencies between vignette and documented source code
